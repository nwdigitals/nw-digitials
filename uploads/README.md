# NW Digitals - Premium Digital Agency Website

This is a premium, modern, conversion-focused single-page marketing website built for **NW Digitals**, a leading digital solutions and web design agency based in South Africa.

## 🚀 Features

- **High-End Corporate Aesthetic:** Impeccable dark-to-light theme transitions, ambient glow effects, glassmorphic navigations, and strong responsive typography.
- **Bespoke Interactive Visuals:**
  - Dynamic responsive portfolio with interactive **Case Studies Modals** showing direct business metrics.
  - Interactive **Pricing Package Connector** (selecting any pricing tier auto-fills the contact form and scrolls to it!).
  - Accordion-based **FAQ Accordion** with smooth rotations.
  - Fully dynamic **Proposal Request Contact Form** with loading spinners and immediate feedback states.
- **Conversion Optimization & Social Proof:**
  - Floating high-impact **WhatsApp Widget** with continuous pulsing notification effect.
  - Urgency ribbon for Q3 project scheduling.
  - Professional client reviews with star ratings and stylized profile grids.
- **Technical Excellence:**
  - **Tailwind CSS CDN & Alpine.js:** Zero clutter, extreme performance, modular elements, and portable file structure.
  - **Custom Scroll Animations:** Performance-optimized Scroll-Reveal script (vanilla Intersection Observer) to create slide-and-fade triggers.
  - **SEO & Accessibility Compliant:** Optimized title tags, meta descriptions, image alt texts, semantic structure, and accessible focus outlines.

## 💻 How To Run

1. Simply double-click `index.html` to open it directly in any modern browser.
2. Alternatively, you can spin up a local development server in the workspace:
   ```bash
   python3 -m http.server 8000
   ```
   Then navigate to `http://localhost:8000` in your web browser.
